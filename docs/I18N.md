# Internationalization Documentation

## Overview
Supports multiple languages using a context-based provider and translation dictionaries.

## Supported Languages
- `en`: English
- `id`: Bahasa Indonesia

## Implementation
- `translations.ts`: Stores the key-value pairs for each language.
- `i18n-context.tsx`: Provides `t` function and `setLanguage` functionality.
- `useI18n`: Custom hook for accessing i18n context in components.

## Usage
```tsx
const { t } = useI18n();
<h1>{t('nav.store')}</h1>
```
