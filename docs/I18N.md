# Internationalization Documentation

## Overview
Supports multiple languages using a context-based provider and translation dictionaries.

## Supported Languages
- `en`: English
- `id`: Bahasa Indonesia

## Implementation
- `translations.ts`: Stores the key-value pairs for each language.
- `i18n-context.tsx`: Provides `t` function and `language` state for client components.
- `get-translations.ts`: Utility for retrieving translations in Server Components.
- `useI18n`: Custom hook for accessing i18n context in client components.

## Usage
### Client Components
```tsx
'use client';
const { t } = useI18n();
<h1>{t('nav.store')}</h1>
```

### Server Components
```tsx
import { getTranslations } from '@/lib/i18n/get-translations';
const { t } = getTranslations(lang);
<h1>{t('nav.store')}</h1>
```
